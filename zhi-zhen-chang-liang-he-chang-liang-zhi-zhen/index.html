指针常量和常量指针

2023-01-08 2 min read

指针常量

指针常量是一个常量，即指针本身是一个常量

const关键字修饰在“ * ”右边就可以表示一个指针常量，并且在声明时就必须初始化，例如： int * const pc = &num

指针常量所指向的地址是不可以修改的，但是可以使用解引用方式修改所指向的具体的数据。

#include <stdio.h>

int main(int argc, char const *argv[])
{
    // 声明两个int类型变量并初始化
    int a = 20, b = 10;
    // 声明一个指针常量ptr并初始化
    int * const ptr = &a;
    // 指针常量所指向的地址是不可以修改的
    ptr = &b; // 错误
    // 指针常量所指向的具体的内容可以通过解引用方式修改
    *ptr = 22; // 正确
    return 0;
}

常量指针

常量指针本质上是一个指针

被const修饰指针变量就是一个常量指针，例如: const int * num = 100

常量指针所指向对象的数据不可以通过解引用的方式修改，但是它指向的地址是可以重新指定的。

#include <stdio.h>

int main(int argc, char const *argv[])
{
   	// 定义一个int类型的变量num
    int num = 10;
    // 定义一个int类型的常量cnum
    const int cnum = 10;
    // 定义一个常量指针, 并初始化为&num;
    const int * pc = &num;
    // 1、常量指针本质是一个指针变量，其指向可以被正常修改
    pc = &cnum; // 正确
    // 2、常量指针所指向的数据不管是常量cnum或是变量num都不可以通过对指针解引用的方式修改
	*pc = 100; // 错误
    return 0;
}
